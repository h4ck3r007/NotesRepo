R [Partial]

Introduction

R system is software environment for statistical computing and graphics. R is open source and is maintained by developers.

Installation

R can be installed from official R website. R provides "R console" to run R commands. There are other ways also to run R code. Here are the other ways to run R code.

  • RStudio : RStudio is free open source IDE for R. It can be downloaded from http://www.rstudio.org
  • As a web application : The rApache software allows to run R into a web application. Details of this can be found from http://biostat.mc.vanderbilt.edu/rapache/
  • As a server : The RServer software allows you to access R from within other applications.

    Coercion

    When you call a function with an argument of the wrong type, R will try to coerce values to a different typeso that function will work.

    Coercion in R

    > x <- c(1,2,3,4,5)
[1] 1 2 3 4 5
> typeof(x)
[1] "double"
> x[2] <- "hat"
> x
[1] "1" "hat" "3" "4" "5"
> typeof(x)
[1] "character"
