React JS : Introduction

ReactJS is a javascript library for building UI for webpages. React is combiner effort from facebook and instagram. React can be used as V (View) under MVC. React uses virtual DOM diff implementation for high performance. The main focus for react is as soon as data changes on required UI portion is updated as quickly as possible.

Running NodeJS server

Let's build hello world page using React JS. This page will simply show the "Hello World" on page. We will first start a server to host our files. We can use NodeJS server provided by react. This code can be downloaded from here To start the server go to the folder and run server.js using node. If required use npm to download express and body-parser modules. Here is commands for doing so.

Code to run Node JS Server

Command to install modules using npm.
> npm install express
> npm install body-parser

Command to start node server
> node server.js
Server started: http://localhost:3000/

This will start a server that will listen on port 3000.

HTML Template for page

Let's build hello world page template. We need react javascript framework files. As react is dependent on jQuery so we also need to add jquery files. We are using JSX (XML-ish syntax in your JavaScript) so we will include javascript file to transport JSX into HTML. JSX is not must but make coding easy to read.

Here is the code for HTML template. We will update the div element with id 'content' using ReactJS.

HTML Template

<html>
  <head>
    <title>Hello React</title>
    <script src="http://fb.me/react-0.12.2.js"></script>
    <script src="http://fb.me/JSXTransformer-0.12.2.js"></script>
    <script src="http://code.jquery.com/jquery-1.10.0.min.js"></script>
  </head>
  <body>
    <div id="content"></div>
    <script type="text/jsx">
      // Your code here
    </script>
  </body>
</html>

React Code

Now we will update the div element to add one H1 tag having text "Hello World". React framework has global javascript object React. This javascript object has method render which takes the html and the element which need to be updated with the html. Code to add h1 tag inside div element using ReactJS is.

Render JavaScript Code

React.render(
        <h1>Hello, world!</h1>,
        document.getElementById('content')
      );

This code need to be added in HTML template we have. Here is complete code for page. When this code will be rendered in browser will show Hello World.

Complete Page Code

<html>
  <head>
    <title>Hello React</title>
    <script src="http://fb.me/react-0.12.2.js"></script>
    <script src="http://fb.me/JSXTransformer-0.12.2.js"></script>
    <script src="http://code.jquery.com/jquery-1.10.0.min.js"></script>
  </head>
  <body>
    <div id="content"></div>
    <script type="text/jsx">
      React.render(
        <h1>Hello, world!</h1>,
        document.getElementById('content')
      );
    </script>
  </body>
</html>
