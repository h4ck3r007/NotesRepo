DOM : Document Object Model

Introduction

DOM or Document Object Model is the tree hierarchy of nodes in HTML document. When a browser renders the HTML it parses the HTML code and convert this in to a tree having hierarchy of nodes. This tree is called DOM tree.

DOM Nodes Type

DOM has different types of nodes like DOCUMENT_NODE (window.document), ELEMENT_NODE (body, div, span etc...). Here are some most common types of nodes.

  • DOCUMENT_NODE eg. window.document
  • ELEMENT_NODE eg. body, span, div, html etc...
  • ATTRIBUTE_NODE eg href
  • TEXT_NODE eg. Whitespaces & carriage returns
  • DOCUMENT_FRAGMENT_NODE
  • DOCUMNET_TYPE_NODE eg. DOCTYPE

    Properties and Methods of Node

    Node is the superclass. Node interface is inhertied by other interfaces like document, HTMLElement etc... Here are some properties and methods.

    Node Properties

    childNodes Return the sub DOM tree having all child nodes.
    firstChild Return the first child node.
    lastChild Return the lsat child node.
    nextSibling Return the next sibling node.
    previousSibling Return the previous sibling node.
    parentNode Return the parent of the current node.
    nodeName Return the node name. For document node it return #document.
    nodeType Return the type of node (Integer value).

    Node Methods

    appendChild() Add new node as child node.
    cloneNode() Clone the current node without its child nodes.
    compareDocumentPosition() ??
    conatins() Return true if node is part of DOM sub tree.
    hasChildNodes() Return true if node is part of DOM sub tree.
    insertBefore() Add node before the current node
    removeChild() Remove the child node from DOM tree.
    replaceChild() Replace the child node to other child node.

    Document Methods to Create the Nodes And Append in DOM

    To create a new element we can use document.createElement() method. To create a text node we can use document.createTextNode().
