Server in Node JS

Running a server in Node JS is very easy.

A Node server is typically created using the createServer method of the http module. Method createServer return the server object. Once you get the server object start listining to some port.

Here is example code to run a simple server using Node JS.

Code to run Node JS Server

var http = require('http');
var server = http.createServer(function(request, response) {

	console.log("Got Request Headers:");
	console.log(request.headers);
	response.writeHead(200, {
		'Content-Type': 'text/plain'
	});
	response.write("PONG");
	response.end();
	
});
server.listen(8080);
