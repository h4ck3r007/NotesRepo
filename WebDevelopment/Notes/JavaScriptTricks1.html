JavaScript Trick #1 : Copy to clipboard

Clipboard Copy

We can run any JavaScript in console but if you are looking to copy the output of JavaScript to other file, JavaScript provide the copy function which takes the String parameter and store this to clipboard.

Copy JavaScript Function

This will copy the String "Hello" on clipboard. You can paste the string in notepad by pressing ctrl+v.

Copy JavaScript Function

copy("Hello");

Clipboard Copy Example

Following code will print the square of one digit numbers.

Clipboard Copy Example Code

var sqrStr="";
for(var i=0;i<10;i++){
	sqrStr += (i + "-->"+ (i*i)+"\n");
};
copy(sqrStr);
